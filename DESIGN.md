# Design System: Stitch Harmony
**Project ID:** GestCave Pro SaaS

## 1. Visual Theme & Atmosphere
The "Stitch Harmony" aesthetic is a modern, professional, and accessible design system. It prioritizes clarity and elegance over raw density. Unlike the previous administrative look, it uses soft geometry and balanced whitespace to create a premium feel.

- **Atmosphere:** Professional, Trustworthy, Modern, Harmonious.
- **Philosophy:** Less is more. Color is used purposefully to guide the user's attention.

## 2. Color Palette & Roles
- **Pristine White (#FFFFFF):** The primary canvas. All pages start with a clean white background for maximum breathing room.
- **Harmony Blue (#1E3A8A):** A professional, deep blue. Used for navigation sidebars, headers, and structural elements. It provides stability without being aggressive.
- **Action Orange (#FF7A00):** The primary action color. Used EXCLUSIVELY for buttons and critical interactive elements. This ensures the user always knows where the primary action is.
- **Slate Grey (#475569):** Used for secondary text and icons to maintain a soft contrast.
- **Soft Border (#E2E8F0):** Subtle dividers and input borders.

## 3. Typography Rules
- **Font Family:** Inter or Public Sans.
- **Headings:** Semi-bold, navy blue (#1E3A8A), natural case (not all-caps).
- **Body Text:** Standard Slate Grey for comfort.
- **Interactive Labels:** Bold, clean, high readability.

## 4. Component Stylings
* **Buttons:** 
    * **Shape:** Generously rounded corners (`rounded-xl` or `rounded-full`).
    * **Color:** Action Orange background, White text.
    * **State:** Subtle hover lift and shadow deepening.
* **Cards/Containers:** 
    * **Shape:** Rounded corners (`rounded-2xl`).
    * **Shadow:** Soft, diffused shadows (`shadow-sm` or `shadow-md`).
    * **Background:** White.
* **Inputs/Forms:** 
    * **Shape:** Rounded corners (`rounded-lg`).
    * **Border:** Thin slate-200, transitioning to Harmony Blue on focus.

## 5. Layout Principles
- **Flow:** Logical, intuitive grouping of elements.
- **Whitespace:** Generous margins to avoid visual fatigue.
- **Visual Hierarchy:** Use size and weight to differentiate information, reserving color for action.
